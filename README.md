## About

BiznisBox is a web application for managing your business. It is a simple and easy to use application that allows you to manage your business in one place. It is a great tool for small and medium-sized businesses.

<img src="https://biznisbox.com/biznisbox_dashboard.png" width="100%" alt="BiznisBox Dashboard" />

## Features

BiznisBox is open source business management software. It is a great tool for small and medium-sized businesses. It is a simple and easy to use application that allows you to manage your business in one place. It integrates with Stripe and PayPal for payments. It also integrates with Nordigen for bank account data.

-   Dashboard with statistics and charts
-   Customers
-   Products
-   Invoices
-   Estimates
-   Payments
-   Online Payments (Stripe and PayPal)
-   Bank Account Data (Nordigen)
-   Transactions
-   Settings
-   Users
-   Roles
-   Permissions
